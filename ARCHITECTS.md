## Key Learnings

### From Session 22 (Implementation)

1. **Memory is Architecture, Not Storage**
   - Threshold-protocols aren't documentationâ€”they're active cognitive state
   - Middleware doesn't log eventsâ€”it enforces governance
   - Tools aren't utilitiesâ€”they're synapses

2. **The Filesystem is a Circuit**
   - BTB action layer = executable substrate
   - Threshold protocols = governance layer
   - MCP tools = synaptic connections
   - Middleware = state memory

3. **Sovereign Stack is Viable**
   - 100% local operation works
   - MLX models + MCP + FastMCP = complete system
   - No cloud dependency needed
   - Full privacy maintained

### From Session 23 (Validation)

1. **Observer Stability Matters**
   - Architecture can be perfect
   - But observer must be stable
   - Reasoning models fail on recursive prompts
   - Tool-calling models succeed

2. **Model Selection is Critical**
   - Hermes-3: Proven stable, no loops
   - DeepSeek-R1: Reasoning interferes with structure
   - Llama-Thinking: Literal interpretation causes infinite loops
   - Rule: If name contains "thinking" or "reasoning", likely incompatible

3. **Build for Real, Not Ideal**
   - Test with what exists
   - Document what works
   - Don't assume capabilities
   - Validate with actual use
